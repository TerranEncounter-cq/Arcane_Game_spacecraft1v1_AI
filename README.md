# Spacecraft1v1
Simple arcane game programmed using Pygame  
  

## Gameplay:
**Yellow player** (left hand side): 
  - Movement: 
    - W - move up
    - A - move left
    - S - move down
    - D - move right
  - Weapon:
    - Minigun - Left Control
    - Rocket - Left Shift

**Red player** (right hand side): 
  - Movement: 
    - ↑ - move up
    - ← - move left
    - ↓ - move down
    - → - move right
  - Weapon:
    - Minigun - Right Control
    - Rocket - Right Shift
